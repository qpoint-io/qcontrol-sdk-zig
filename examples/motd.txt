
  =============================================
   qcontrol SDK for Zig - Demo Environment
  =============================================

  Build a plugin:

    cd examples/file-logger
    make build                    # builds .so (shared library)
    make dist                     # builds .o  (object file for bundling)

  Build all example plugins:

    cd examples
    make build

  Build plugins by type:

    make build-file              # file plugin shared libraries
    make build-exec              # exec plugin shared libraries
    make build-net               # net plugin shared libraries

  Bundle plugins into a single library:

    cd examples
    qcontrol bundle -c bundle.toml

  Bundle plugins by type:

    make bundle-file             # zig-file-plugins.so
    make bundle-exec             # zig-exec-plugins.so
    make bundle-net              # zig-net-plugins.so

  Run a command with qcontrol instrumentation:

    qcontrol wrap --bundle <bundle.so> -- <command>

  Example - instrument 'cat' with bundled plugins:

    cd examples
    qcontrol bundle -c bundle.toml
    qcontrol wrap --bundle zig-plugins.so -- cat /etc/hostname

  Useful commands:

    qcontrol --help               # full CLI help
    qcontrol wrap --help          # instrumentation options
    qcontrol bundle --help        # bundling options
